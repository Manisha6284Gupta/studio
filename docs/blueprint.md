# **App Name**: CivicConnect

## Core Features:

- User Authentication & Profiles: Enable citizens to register and log in via phone/email with password/OTP. Departments and Control Room users also have login. Includes password reset and user profile management.
- Complaint Submission: Citizens can submit new complaints including title, a description using mic-to-text input, image uploads, and geographical location. An application number is automatically generated.
- AI-Powered Complaint Routing & Deadlines: An AI tool analyzes the complaint description and title to assist in categorizing the problem type, which in turn facilitates routing the complaint to the relevant department and helps in setting initial complaint deadlines.
- Citizen Complaint Tracking: Citizens can view their submitted complaints on a dashboard, monitor status, see full details, edit (if allowed by status), reapply, delete, and provide feedback on resolved issues.
- Departmental Workflow Dashboard: Departments can view and manage complaints assigned to them, update status, access full complaint details, and monitor overdue complaints within their specific area.
- Control Room Administration: Control Room users have all departmental features, plus the ability to view all complaints department-wise, monitor department performance through visualizations, and reassign complaints as needed.
- Geospatial & Data Visualization: Integrate interactive maps to display complaint locations for citizens (their own), departments (assigned), and control room (all). Provide graphs and visualizations for complaint status, resolution times, and departmental performance.

## Style Guidelines:

- Primary color: A deep, professional blue-cyan (#1F89AD), symbolizing trust and clarity in civic engagement.
- Background color: A very light, almost off-white blue-gray (#F0F4F5), for an open and approachable aesthetic in a light theme.
- Accent color: A vibrant medium blue (#2653D9), used for interactive elements and highlights to provide clear contrast and emphasize important actions.
- Headlines font: 'Space Grotesk' (sans-serif), for a modern and precise heading style. Body text font: 'Inter' (sans-serif), for clear readability in longer descriptions and dashboard information.
- Utilize clear and intuitive line-based icons throughout the application to represent different complaint categories, statuses, and user actions, maintaining a contemporary feel.
- Implement a responsive grid layout to ensure optimal viewing across various devices. Dashboards will feature a clear separation of content sections, prioritizing key information and accessibility to primary actions, with complaint locations integrated into map views.
- Incorporate subtle and smooth transitions for page navigation, status updates, and form submissions to enhance the user experience without being distracting.
